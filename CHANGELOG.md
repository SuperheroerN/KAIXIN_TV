# 更新日志

本文档记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-23

### 🎉 首次发布

基于 OuonnkiTV 的完整重构版本。

### ✨ 新增功能

#### 首页推荐
- 集成豆瓣电影 API
- 支持多种分类（热门、最新、经典等）
- 电影类型分类（动作、喜剧、爱情等）
- 地区分类（华语、欧美、韩国、日本）
- 电视剧分类（美剧、英剧、韩剧等）
- 点击推荐内容触发搜索

#### 主题系统
- 手动切换深色/浅色模式
- 全局背景渐变效果
- 白天模式：天空蓝渐变
- 夜间模式：深蓝紫渐变
- 主题状态持久化

#### 访问控制
- 可选的密码保护功能
- 防暴力破解机制
  - 3次失败 → 锁定30秒
  - 4次失败 → 锁定1分钟
  - 5次失败 → 锁定2分钟
  - 6次及以上 → 锁定时间递增
- 实时倒计时显示
- 失败次数提示

#### 搜索优化
- 多源聚合搜索
- 按源优先级排序结果
- 自动去重
- 增量渲染
- 搜索结果缓存
- 支持取消搜索

#### 视频源管理
- 环境变量统一配置
- 只读显示模式
- 跨浏览器同步
- 支持多个视频源
- 自动检测不支持搜索的源

#### 代理系统
- 统一代理中间件
- 图片代理支持
- 防盗链处理
- 豆瓣 API 特殊处理
- 缓存优化

#### UI/UX 改进
- 响应式设计
- 移动端适配
- 加载动画
- 错误提示
- Toast 通知
- 下拉菜单分组
- 滚动优化

### 🔧 技术改进

- 升级到 React 18.3
- 升级到 TypeScript 5.6
- 升级到 Vite 6.0
- 代码分割优化
- 懒加载实现

### 🎨 品牌定制

- 自定义 Favicon

### 📝 文档

## 版本说明

### 版本号格式

`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型

- **新增 (Added)**：新功能
- **变更 (Changed)**：现有功能的变更
- **弃用 (Deprecated)**：即将移除的功能
- **移除 (Removed)**：已移除的功能
- **修复 (Fixed)**：Bug 修复
- **安全 (Security)**：安全相关的修复
